# Contributing

This project prioritizes fixes for critical errors, compatibility problems, and unintended behavior over speculative feature expansion.

When opening an issue or submitting a pull request, include:

- Clear reproduction steps and the expected result
- The Windows version used for testing
- Relevant structural details of the affected PNG or Sprite file
- Any known compatibility or functionality impact
- A concise explanation of the change and its purpose

Do not add original game files or assets with unclear redistribution rights to the repository.

Redistributed or modified versions must follow the attribution, base-version identification, and change-disclosure requirements in [`LICENSE`](LICENSE).
