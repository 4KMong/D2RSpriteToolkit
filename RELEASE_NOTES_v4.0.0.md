# D2R Sprite Toolkit v4.0.0

## Highlights

- Low-resolution PNG generation and `.lowend.png` output
- Sprite preview, metadata inspection, and PNG conversion
- PNG-to-Sprite conversion with static output or frame-template preservation
- Batch file renaming tools
- Korean and English user interfaces
- Dedicated system-tray icon
- Portable single-executable distribution

## Installation

1. Download `D2RSpriteToolkit_v4.0.0_Windows.zip`.
2. Extract the ZIP file.
3. Run `D2RSpriteTK.exe`.

## Notes

- Windows only
- No installer is required
- Windows SmartScreen may display a warning because the executable is not code-signed
- Back up original files before processing them
- This is an unofficial fan-made utility and is not an official Blizzard Entertainment product
